# Portfolio Website

## Project Overview
This project is a personal portfolio website developed using HTML and CSS. It showcases my academic background, technical and non-technical skills, and projects I have worked on. The website is designed with a modern UI, including a full-screen background video, animated elements, and responsive layout.

The main purpose of this project is to present my profile in a structured and visually appealing manner while demonstrating front-end development skills.


## Technologies Used
* HTML5
* CSS3

## Setup Instructions

1. Download or clone the repository:
   ```
   git clone https://amrithapn.github.io/portfolio-website/
   ```
2. Open the project folder.
3. Ensure all files (images and video) are in the correct directory as shown above.

## How to Run the Project
1. Open the `index.html` file in any web browser.
2. The website will load with all sections including Home, About, Education, Projects, Skills, and Contact.


## Deployment
This project can be hosted using GitHub Pages:

1. Go to repository settings
2. Navigate to **Pages**
3. Select branch: `main`
4. Save and access the live link


## Features

* Full-screen background video
* Smooth navigation between sections
* Responsive design for different screen sizes
* Structured sections: About, Education, Projects, Skills, Contact
